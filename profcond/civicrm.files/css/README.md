This directory is for site-admin-defined CSS files that may
be injected by the com.joineryhq.profcond extension.

Files placed here (other than this README file -- which may be 
overrwitten on extension upgrade) are owned and maintained by
the site administrator, not by the extension.

You may safely add, edit, or remove files in this directory
(but it's recommended to leave this README file untouched).
